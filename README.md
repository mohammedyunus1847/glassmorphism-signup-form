# Modern Sign Up Page

A clean and modern Sign Up page created using HTML5 and CSS3. This project was built to practice frontend development, responsive layouts, and modern UI design techniques.

## Features

- Modern glassmorphism design
- Responsive centered layout
- HTML5 form validation
- Custom Google and Apple sign-up buttons
- Background glow effects using CSS pseudo-elements
- Clean and organized code

## Technologies Used

- HTML5
- CSS3
- Flexbox
